1. To run it, first install selenium using 'pip install selenium'
2. Update chrome to newest version
3. Download latest chromedriver (also provided in the zip) and place it in "C:\Program Files\Google\Chrome\Application\chromedriver.exe" or in the chrome's directory if chrome is not installed in default location
4. Type 'python script.py' in terminal 
5. A new window will open with whatsapp web. Login to your id by scanning QR code. Then come back to terminal and press Enter
6. Enter relevant information to create the group as asked by the prompts
7. Now the group has been created
